ProGo Pro-Blogger - Make your blog work for you! Create a better converting blog with New layout options, Calls-To-Action, and more.

== Description ==

ProBlogger plugin adds new layout and functionality to your ProGo theme.

Features include:

*   Bootstrap Jumbotron area to draw attention
*   New Top Menu navbar for quick action items and widgets
*   Masthead area to advertise your logo or calls to action
*   Call-To-Action sidebar widget that supports custom forms and Gravity Forms

== Installation ==

1. Upload the contents of `progo-problogger.zip` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to the WordPress Customizer panel to start customizing your blog
